# sd-webui-open-in-editor
This extension allows you to send images generated in Stable Diffusion WebUI (AUTOMATIC1111) directly to external image editing software, such as Photoshop, with a single click.
https://youtu.be/oKeBXXuY92o
## Installation Instructions
1. Open the `Extensions` tab in the WebUI
2. Open the `Install from URL` tab
3. Enter the URL for this repository in the `URL for extension's git repository` field:
   `https://github.com/kiramesu/sd-webui-open-in-editor`
4. Click the `Install` button
5. In the `Installed` tab, click `Apply and restart UI`

## How to Use
1. Open `Settings` > `Paint software`.

2. Enter the full path to the `.exe` file of the painting software you want to use, then save the settings.
   *(Example: `C:\Program Files\Adobe\Adobe Photoshop 2024\Photoshop.exe`)*

3. Click the “🖍” button added below the gallery to instantly open the currently displayed image in the specified software.
